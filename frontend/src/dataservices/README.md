# Dataservices (data layer)

**Category:** API integration  
**Purpose:** HTTP clients, env config, and types shared with the backend.

| Subfolder | Purpose |
|-----------|---------|
| `config/` | Environment constants (`API_BASE_URL`) |
| `api/` | Axios client and per-resource API modules |
| `types/` | TypeScript interfaces matching backend DTOs |

**Do not** add JSX, page layout, or CSS here.
